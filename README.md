# students_portal
# 🎓 Student Academic Portal

A full-stack web application built using Flask and PostgreSQL for managing student academic records.

## 🚀 Features
- Student management (CRUD)
- Grade management
- Analytics dashboard with charts
- PDF report generation
- PostgreSQL database integration

## 🛠 Tech Stack
- Python (Flask)
- PostgreSQL
- ReportLab
- Matplotlib / Chart.js
